## Install

```bash
npm i -g kenote-upng
```

## Usages

```bash

  Usage: node-upng [command] [options]


  Options:

    -V, --version  output the version number
    -h, --help     output usage information


  Commands:

    add     add one custom project
    start   run one project ...
```
